# Hi
A chat application, based near the support of people shown on the map(LBS)

##简介
- 一个基于Bmob的社交即时聊天应用

- 仿最新微信底部导航渐变效果

- 摇一摇查找附近的人，基于百度地图将附近的人显示在地图上

- 通过玩猜拳小游戏添加匿名好友

- 黑名单，发送语音，表情，图片，位置等
